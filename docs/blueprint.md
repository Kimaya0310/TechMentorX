# **App Name**: Social Bridge

## Core Features:

- User Authentication and Roles: Securely authenticate users (donors, volunteers, NGOs, beneficiaries, admins) with role-based access control using Firebase Authentication.
- Donation Management: Enable donors to contribute items, track donation requests based on location, and monitor the donation lifecycle. Incorporate photo-based verification for transparency.
- Volunteer Coordination: Assign tasks to volunteers, provide task checklists, and enable route support for efficient volunteer efforts.
- Impact Tracking Dashboard: Display key metrics such as total donations, people helped, and items distributed through a user-friendly impact dashboard.
- Gamification and Rewards: Implement a points system, badges, leaderboards, and auto-generated certificates to motivate donors and volunteers.
- AI-Powered Welfare Scheme Recommendation Tool: An AI-powered chatbot that suggests relevant welfare schemes based on user inputs like age, income, and disability status.
- Admin Control Panel: Admin dashboard to verify organizations, manage welfare schemes, monitor platform activity, and generate comprehensive analytics reports.

## Style Guidelines:

- Primary color: A professional and trustworthy dark blue (#003366).
- Secondary color: A calm and neutral light gray (#F5F5F5) for backgrounds and content containers.
- Accent color: A subtle teal (#008080) for interactive elements and highlights, providing a touch of vibrancy without being overwhelming.
- Font pairing: 'Roboto' for body text (sans-serif, clean and readable) and 'Montserrat' for headlines (sans-serif, modern and bold).
- Simple, line-based icons to represent various categories and actions.
- Card-based layout to organize information in a clean and structured manner.
- Subtle hover effects to provide visual feedback and enhance interactivity.
